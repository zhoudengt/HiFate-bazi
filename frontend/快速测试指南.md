# 前端 Demo 快速测试指南

## 🚀 快速开始（3 步）

### 步骤 1：启动后端服务

```bash
cd /Users/zhoudt/Downloads/project/HiFate-bazi
python server/start.py
```

等待看到：`✓ 服务器已启动（端口 8001）`

### 步骤 2：启动前端服务器

**方式 A：使用启动脚本（推荐）**

```bash
cd frontend
./start.sh
```

**方式 B：手动启动**

```bash
cd frontend
python3 -m http.server 8080
```

### 步骤 3：打开浏览器

访问：`http://127.0.0.1:8080`

## 🐛 常见问题

### 问题 1：页面空白

**解决：** 检查浏览器控制台（F12），查看是否有 JavaScript 错误。

### 问题 2：API 调用失败

**解决：**
1. 确认后端服务已启动（`http://127.0.0.1:8001`）
2. 检查 `config.js` 中的 API 地址是否正确
3. 查看浏览器 Network 标签，检查请求和响应

### 问题 3：CORS 错误

**解决：** 必须使用本地服务器（`start.sh`），不能直接打开 HTML 文件。

### 问题 4：登录失败

**解决：**
1. 检查后端服务是否运行
2. 检查用户名密码是否正确（默认：admin/admin123）
3. 查看后端日志：`logs/web_app_8001.log`
