# 事业财富 - 提示词规则说明

提示词在 **Coze（或百炼）** 的「事业财富」Bot 的 **System Prompt** 里改。

---

## 去掉考证、考编、具体城市（全球客户）

### 1. 第 6 条里补一句

原有：「禁止推荐具体城市，方位建议只说方向……」

改为（或在其后补一句）：

```
禁止推荐具体城市，方位只说方向（如东南方、南方），不提及具体城市名。客户为全球用户。
```

### 2. 核心约束里新增一条

在核心约束中增加一条（编号顺延即可）：

```
禁止推荐考证、考编等具体考试或编制路径；只分析事业特质与行业方向，不引导特定考试。
```

你已有「方位只说方向、不推荐具体城市」，只需补「客户为全球用户」并新增「不推荐考证、考编」即可。
