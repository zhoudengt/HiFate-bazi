## 使用说明

1. **开发时**：参考 `.cursorrules` 获取核心规范和快速参考
2. **深入学习**：查看本目录下的详细规范文档，了解完整的开发流程、代码示例和检查清单
3. **规范更新**：所有规范更新应该先更新详细文档，然后同步更新 `.cursorrules` 中的核心要点
