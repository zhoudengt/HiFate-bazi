# FateTell 风格设计说明

## 🎨 设计理念

参考 FateTell 网页版的现代化设计风格，采用深色主题、渐变效果和流畅动画，打造专业、优雅的用户体验。

## ✨ 主要特性

### 1. 深色主题
- **主背景**: 深蓝黑色 (`#0f172a`)
- **卡片背景**: 深灰蓝色 (`#1e293b`)
- **文字**: 浅色系，确保良好的对比度

### 2. 渐变色彩
- **主渐变**: 靛蓝 → 紫色 → 粉红 (`#6366f1 → #8b5cf6 → #ec4899`)
- **应用场景**: 标题、按钮、强调元素

### 3. 现代化交互
- **悬停效果**: 卡片上浮、阴影增强
- **动画过渡**: 使用 `cubic-bezier` 缓动函数
- **视觉反馈**: 边框高亮、渐变条显示

### 4. 精致细节
- **自定义滚动条**: 细窄、带渐变色
- **加载动画**: 旋转的加载指示器
- **徽章设计**: 圆角、渐变背景

## 🎯 设计元素

### 颜色系统

```css
/* 主色调 */
--primary-color: #6366f1      /* 靛蓝色 */
--secondary-color: #8b5cf6    /* 紫色 */
--accent-color: #ec4899      /* 粉红色 */

/* 背景色 */
--bg-primary: #0f172a         /* 深蓝黑 */
--bg-secondary: #1e293b       /* 深灰蓝 */
--bg-card: #1e293b            /* 卡片背景 */

/* 文字色 */
--text-primary: #f1f5f9       /* 主文字 */
--text-secondary: #cbd5e1     /* 次要文字 */
--text-muted: #94a3b8         /* 弱化文字 */
```

### 组件样式

#### 卡片 (Card)
- 深色背景 + 边框
- 悬停时上浮 + 阴影增强
- 顶部渐变条（悬停时显示）

#### 按钮 (Button)
- 渐变背景
- 波纹效果（悬停时）
- 平滑过渡动画

#### 表单 (Form)
- 深色输入框
- 聚焦时边框高亮 + 阴影
- 清晰的标签和提示

#### 运势项 (Fortune Item)
- 左侧渐变条指示器
- 悬停时右移 + 边框高亮
- 激活状态：背景色 + 发光效果

## 🚀 使用方法

### 1. 已自动应用
所有主要页面已自动引入 `fatetell-style.css`，刷新页面即可看到效果。

### 2. 自定义调整
如需调整颜色或样式，编辑 `css/fatetell-style.css` 中的 CSS 变量：

```css
:root {
    --primary-color: #6366f1;  /* 修改主色调 */
    --bg-primary: #0f172a;     /* 修改背景色 */
    /* ... 其他变量 */
}
```

### 3. 响应式设计
- **桌面端**: 多列网格布局
- **平板端**: 自适应列数
- **移动端**: 单列布局

## 📱 响应式断点

- **大屏** (> 1200px): 3列网格
- **中屏** (768px - 1200px): 2列网格
- **小屏** (< 768px): 1列布局

## ✨ 动画效果

### 淡入动画
```css
.fade-in {
    animation: fadeIn 0.5s ease-out;
}
```

### 加载动画
```css
.loading::after {
    animation: spin 1s linear infinite;
}
```

### 悬停效果
- 卡片上浮: `translateY(-4px)`
- 阴影增强: `shadow-md → shadow-xl`
- 边框高亮: 主色调边框

## 📝 注意事项

1. **样式优先级**: `fatetell-style.css` 在 `style.css` 之后引入，会覆盖部分原有样式
2. **保留原有功能**: 所有原有功能保持不变，只改变视觉样式
3. **性能优化**: 使用 CSS 变量和硬件加速的动画，性能良好
