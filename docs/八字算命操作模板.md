## 快速开始

### 第一步：准备信息

需要收集以下信息：
- **出生日期**：阳历日期（格式：1990-05-15）
- **出生时间**：具体时间（格式：14:30）
- **性别**：男 或 女

### 第二步：调用系统

**方式一：使用网页界面**
1. 打开浏览器访问：`http://127.0.0.1:8001/docs`
2. 找到 `/api/v1/bazi/rules/match` 接口
3. 填写信息并提交

**方式二：使用命令行（curl）**
```bash
curl -X POST http://127.0.0.1:8001/api/v1/bazi/rules/match \
  -H "Content-Type: application/json" \
  -d '{
    "solar_date": "1990-05-15",
    "solar_time": "14:30",
    "gender": "male",
    "rule_types": ["marriage_ten_gods", "marriage_element", "rizhu_gender_dynamic"],
    "include_bazi": true
  }'
```

### 第三步：查看结果

系统会返回：
- 完整的八字信息（四柱、神煞、十神等）
- 匹配到的规则列表
- 每条规则的详细分析内容

## 八字基础知识

### 什么是八字？

八字，又称四柱，由以下四部分组成：

```
年柱：出生年份的天干地支
月柱：出生月份的天干地支
日柱：出生日期的天干地支
时柱：出生时辰的天干地支
```

**示例：**
```
年柱：庚午（1990年）
月柱：辛巳（5月）
日柱：甲子（15日）
时柱：辛未（14:30）
```

### 天干地支表

**十天干：**
- 甲、乙、丙、丁、戊、己、庚、辛、壬、癸

**十二地支：**
- 子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

### 四柱的含义

| 柱位 | 代表含义 | 重要性 |
|------|---------|--------|
| **年柱** | 祖上、父母、早年运势 | ⭐⭐⭐ |
| **月柱** | 兄弟姐妹、青年运势 | ⭐⭐⭐⭐ |
| **日柱** | 本人、配偶、中年运势 | ⭐⭐⭐⭐⭐ |
| **时柱** | 子女、晚年运势 | ⭐⭐⭐ |

### 十神（主星）

十神是根据日干与其他天干的关系确定的：

| 十神 | 含义 | 代表 |
|------|------|------|
| **比肩** | 同类相助 | 朋友、同事 |
| **劫财** | 同类相争 | 竞争、破财 |
| **食神** | 我生之物 | 才华、口福 |
| **伤官** | 我生之物（偏） | 才华、任性 |
| **偏财** | 我克之物（偏） | 横财、情人 |
| **正财** | 我克之物 | 正财、妻子 |
| **七杀** | 克我之物（偏） | 压力、小人 |
| **正官** | 克我之物 | 官运、丈夫 |
| **偏印** | 生我之物（偏） | 偏业、继母 |
| **正印** | 生我之物 | 学业、母亲 |

### 神煞

神煞是八字中的特殊标记，常见的有：

| 神煞 | 含义 | 影响 |
|------|------|------|
| **天乙贵人** | 贵人相助 | 遇事有人帮助 |
| **天德贵人** | 品德高尚 | 性格善良，少灾 |
| **月德贵人** | 月德庇护 | 有贵人相助 |
| **桃花** | 异性缘 | 感情丰富 |
| **驿马** | 走动多 | 经常外出 |
| **羊刃** | 刚强 | 性格刚烈 |
| **华盖** | 艺术天赋 | 有艺术才能 |

## 如何使用系统

### 1. 基础八字计算

**请求示例：**
```json
{
  "solar_date": "1990-05-15",
  "solar_time": "14:30",
  "gender": "male"
}
```

**返回内容：**
- 四柱信息（年柱、月柱、日柱、时柱）
- 十神统计
- 五行统计
- 神煞列表
- 纳音信息

### 2. 规则匹配

**请求示例：**
```json
{
  "solar_date": "1990-05-15",
  "solar_time": "14:30",
  "gender": "male",
  "rule_types": [
    "marriage_ten_gods",      // 婚姻十神规则
    "marriage_element",       // 婚姻五行规则
    "marriage_day_pillar",    // 婚姻日柱规则
    "rizhu_gender_dynamic"    // 日柱性别分析
  ],
  "include_bazi": true
}
```

**返回内容：**
- 匹配到的规则列表
- 每条规则的详细分析

### 3. 完整分析（包含大运流年）

**请求示例：**
```json
{
  "solar_date": "1990-05-15",
  "solar_time": "14:30",
  "gender": "male",
  "current_time": "2025-01-15 10:00"
}
```

**返回内容：**
- 基础八字信息
- 大运序列
- 流年序列
- 当前大运流年信息

## 解读规则结果

### 规则结果格式

系统返回的规则结果包含以下信息：

```json
{
  "rule_id": "MARR_001",
  "rule_code": "MARR_001",
  "rule_name": "规则名称",
  "rule_type": "marriage_ten_gods",
  "priority": 100,
  "content": {
    "type": "description",
    "items": [
      {"type": "description", "text": "规则描述内容1"},
      {"type": "description", "text": "规则描述内容2"}
    ]
  },
  "description": "规则描述"
}
```

### 如何解读

1. **查看规则类型**：了解这是哪类规则（婚姻、桃花等）
2. **阅读规则内容**：查看具体的命理分析
3. **结合八字信息**：将规则内容与八字信息结合理解

### 规则类型说明

| 规则类型 | 说明 | 关注点 |
|---------|------|--------|
| `marriage_ten_gods` | 婚姻十神规则 | 十神对婚姻的影响 |
| `marriage_element` | 婚姻五行规则 | 五行对婚姻的影响 |
| `marriage_day_pillar` | 婚姻日柱规则 | 日柱对婚姻的影响 |
| `marriage_deity` | 婚姻神煞规则 | 神煞对婚姻的影响 |
| `taohua_general` | 桃花通用规则 | 桃花运相关 |
| `rizhu_gender_dynamic` | 日柱性别分析 | 日柱+性别的综合分析 |

```

3. 获取结果
   ├─ 八字信息
   ├─ 匹配的规则
   └─ 详细分析

4. 解读分析
```

## 解读技巧

### 1. 优先级排序

规则按优先级排序，优先级高的规则更重要：
- **优先级 100+**：核心规则，必须关注
- **优先级 50-99**：重要规则，重点关注
- **优先级 < 50**：参考规则，辅助分析

### 2. 综合多个规则

不要只看一条规则，要综合多条规则：
- 日柱分析 + 婚姻规则 = 完整的婚姻分析
- 十神分析 + 神煞分析 = 完整的性格分析

### 3. 注意规则类型

不同规则类型关注点不同：
- `marriage_*`：关注婚姻
- `taohua_*`：关注桃花
- `rizhu_gender_dynamic`：全面分析

### 4. 结合八字信息

将规则内容与八字信息结合：
- 规则说"正官多"，查看八字中正官的数量
- 规则说"有桃花"，查看八字中桃花的位置

## 常见问题

### Q1：如何理解规则内容？

**A：** 规则内容是对命理的专业分析，需要结合八字信息理解。建议：
1. 先看日柱分析（最全面）
2. 再看具体规则（针对性分析）
3. 综合多个规则得出完整结论

### Q2：规则太多，如何选择？

**A：** 系统会自动匹配相关规则，你可以：
1. 不指定 `rule_types`，系统会匹配所有相关规则
2. 指定 `rule_types`，只匹配特定类型的规则

### Q3：如何判断规则是否准确？

**A：** 规则基于传统命理学，建议：
1. 综合多个规则判断
2. 结合实际情况验证
3. 关注优先级高的规则

### Q4：如何给客户解释？

**A：** 建议流程：
1. 先解释八字基本信息
2. 再解释匹配的规则
3. 最后给出综合建议
