# Coze Bot 配置 GPT 模型指南

## 📋 概述

在 Coze 平台，模型的选择是在 Bot 配置中完成的，而不是在 API 调用时指定。本指南将说明如何在 Coze 平台为您的 Bot 选择或更换 GPT 模型。

## 🚀 配置步骤

### 步骤1：访问 Coze 平台

1. 打开浏览器，访问：**https://www.coze.cn**
2. 登录您的账号

### 步骤2：进入 Bot 编辑页面

1. 在 Coze 平台首页，找到您要配置的 Bot
2. 点击 Bot 名称或"编辑"按钮，进入 Bot 编辑页面
3. 或者直接访问 Bot 编辑页面（URL 中包含 Bot ID）

### 步骤3：找到模型配置选项

在 Bot 编辑页面，查找以下位置之一：

#### 方式1：模型选择器（最常见）

1. 在 Bot 编辑页面的**左侧菜单**或**顶部工具栏**中
2. 找到 **"模型"**、**"Model"** 或 **"选择模型"** 选项
3. 点击打开模型选择器

#### 方式2：高级设置

1. 在 Bot 编辑页面，找到 **"高级设置"** 或 **"Advanced Settings"**
2. 展开高级设置
3. 找到 **"模型配置"** 或 **"Model Configuration"** 选项

#### 方式3：对话设置

1. 在 Bot 编辑页面，找到 **"对话设置"** 或 **"Chat Settings"**
2. 在对话设置中，找到 **"模型"** 选项

### 步骤4：选择 GPT 模型

在模型选择器中，您会看到可用的模型列表，包括：

#### 免费版可用模型：
- **GPT-3.5-turbo**（默认，免费）
- **GPT-4o**（有限次数，通常每日 5 次）

#### 付费版可用模型：
- **GPT-4o**（更多次数）
- **GPT-4-turbo**
- **GPT-4**
- **GPT-3.5-turbo**（更多次数）

#### 选择步骤：

1. **点击模型选择器**，打开模型列表
2. **选择您想要的 GPT 模型**（例如：GPT-4o）
3. **确认选择**（某些平台可能需要点击"保存"或"应用"）

### 步骤5：保存配置

1. 选择模型后，**点击"保存"**或"发布"按钮
2. 等待保存完成（通常几秒钟）
3. 确认保存成功（页面会显示"保存成功"或类似提示）

## 🔍 验证模型配置

### 方法1：在 Coze 平台测试

1. 在 Bot 编辑页面，找到 **"测试"** 或 **"预览"** 按钮
2. 输入测试问题，例如："你好，请介绍一下你自己"
3. 查看回复，确认使用的是您选择的模型

### 方法2：通过 API 测试

运行项目中的测试脚本：

```bash
# 测试 Coze API 调用
python scripts/tools/test_coze_api_direct.py
```

如果配置成功，API 调用会使用您选择的模型。

### 方法3：查看 API 响应

在 API 响应中，可能会包含模型信息（取决于 Coze API 的实现）：

```json
{
  "messages": [
    {
      "role": "assistant",
      "content": "AI 分析结果...",
      "model": "gpt-4o"  // 如果 API 返回模型信息
    }
  ]
}
```

## ⚙️ 模型选择建议

### 根据使用场景选择：

| 场景 | 推荐模型 | 原因 |
|------|---------|------|
| **日常对话** | GPT-3.5-turbo | 速度快，成本低 |
| **专业分析** | GPT-4o | 分析能力强，理解深入 |
| **复杂推理** | GPT-4-turbo | 推理能力强 |
| **高并发场景** | GPT-3.5-turbo | 响应快，支持更多并发 |

### 根据预算选择：

| 预算 | 推荐模型 | 说明 |
|------|---------|------|
| **免费版** | GPT-3.5-turbo | 免费，但 GPT-4o 每日限 5 次 |
| **付费版（$9/月）** | GPT-4o | 更多 GPT-4o 调用次数 |
| **付费版（$39/月）** | GPT-4o / GPT-4-turbo | 最多调用次数，支持高并发 |

## 📝 注意事项

### 1. 模型切换影响

- **切换模型后立即生效**：不需要重启服务
- **API 调用会自动使用新模型**：无需修改代码
- **不同模型响应时间不同**：GPT-4o 通常比 GPT-3.5-turbo 慢

### 2. 免费版限制

- **GPT-4o 每日限 5 次**：超过后会自动降级到 GPT-3.5-turbo
- **建议设置默认模型为 GPT-3.5-turbo**：避免频繁降级

### 3. 付费版优势

- **更多 GPT-4o 调用次数**：Premium 版有更多额度
- **支持高并发**：Premium Plus 版支持更高并发
- **无降级风险**：不会因为额度用完而降级

### 4. 模型性能对比

| 模型 | 响应速度 | 分析能力 | 成本 |
|------|---------|---------|------|
| GPT-3.5-turbo | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 低 |
| GPT-4o | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 中 |
| GPT-4-turbo | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 高 |

## 🔧 常见问题

### Q1: 找不到模型选择器？

**A**: 不同版本的 Coze 平台界面可能不同，尝试：
- 查看 Bot 编辑页面的所有菜单项
- 查看"高级设置"或"对话设置"
- 查看 Coze 平台帮助文档

### Q2: 选择了 GPT-4o 但还是使用 GPT-3.5？

**A**: 可能原因：
- 免费版额度已用完（每日限 5 次）
- 需要升级到付费版
- 模型配置未保存成功

**解决**：
1. 检查 Coze 平台的使用量统计
2. 确认模型配置已保存
3. 考虑升级到付费版

### Q3: 如何查看当前使用的模型？

**A**: 
1. 在 Coze 平台，查看 Bot 配置页面
2. 查看 API 响应（如果包含模型信息）
3. 通过测试对话，观察响应质量（GPT-4o 通常更详细）

### Q4: 切换模型后需要重启服务吗？

**A**: **不需要**。模型配置在 Coze 平台，切换后立即生效，API 调用会自动使用新模型。

### Q5: 可以在代码中指定模型吗？

**A**: **不可以**。Coze API 不支持在请求中指定模型，模型选择必须在 Coze 平台的 Bot 配置中完成。
