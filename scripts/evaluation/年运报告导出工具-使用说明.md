# 年运报告导出工具 - 使用说明

## 一、安装 Python

**Mac：**
```bash
brew install python3
```

**Windows（管理员 CMD）：**
```cmd
winget install Python.Python.3.12
```

---

## 二、安装 pip 和依赖

**Mac：**
```bash
pip3 install aiohttp pandas openpyxl
```

**Windows：**
```cmd
pip install aiohttp pandas openpyxl
```

---

## 三、脚本放哪、在哪执行

| 内容 | 说明 |
|------|------|
| **脚本放哪** | 任意目录，如 `~/Desktop/年运导出/`（Mac）或 `C:\年运导出\`（Windows） |
| **在哪执行** | 在**脚本所在目录**打开终端，执行下面的命令 |

---

## 四、示例

假设：
- 脚本在 `~/Desktop/年运导出/annual_report_exporter.py`
- Excel 在 `~/Desktop/年运报告.xlsx`

**Mac：**
```bash
cd ~/Desktop/年运导出
python3 annual_report_exporter.py --input ~/Desktop/年运报告.xlsx --year 2025 --concurrency 3 --verbose
```

**Windows：**
```cmd
cd C:\年运导出
python annual_report_exporter.py --input "C:\Users\你的用户名\Desktop\年运报告.xlsx" --year 2025 --concurrency 3 --verbose
```

---

## 五、Excel 格式

A 列 USER_ID | B 列 姓名 | C 列 生辰 | D 列 性别 | E 列 年运报告（留空，脚本会写入）

生辰示例：`1987年1月7日 9:15`，性别：`男`/`女`。
